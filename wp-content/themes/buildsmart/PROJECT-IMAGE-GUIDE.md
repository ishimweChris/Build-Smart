# How to Add Images to Projects

## Method 1: Add Featured Image (Main Project Image)

1. Go to **Dashboard** → **Projects** → **All Projects**
   - URL: http://localhost/build/wp-admin/edit.php?post_type=project

2. Click on any project name to edit it

3. On the right sidebar, look for **"Featured Image"** box
   - If you don't see it, scroll down on the right side
   - Or click the gear icon (⚙️) in top right, then enable "Featured Image" in preferences

4. Click **"Set featured image"**

5. Select an image from your uploaded media (or upload new ones)

6. Click **"Set featured image"** button

7. Click **"Update"** to save the project

## Method 2: Add Images Inside Project Content

1. Edit the project (same as above)

2. Click inside the content editor where you want to add images

3. Click the **"+"** button to add a block

4. Select **"Image"** block

5. Upload or select images from media library

6. Add caption if desired

7. Click **"Update"** to save

## Quick Links

## Tips

- **Featured images** appear in project listings and at the top of project pages
- Use high-quality images (recommended: 1200x800px or larger)
- Add descriptive alt text for SEO and accessibility
- You can add multiple images inside the content area
- Images will automatically resize for mobile devices
